# layui-chrome

> 一款layui fly社区chrome的小插件





## 目录结构

```
  ├─dist                  //chrome插件
  │  │─layui-chrome.crx     //chrome可安装插件
  │  │─layui-chrome.pem     //chrome插件秘钥
  ├─src                  //未压缩前目录
  │  │─static               //静态资源  
  │  │─layui.png            //图标资源
  │  │─manifest.json        //配置文件
  │  │─popup.html           //弹出页面
  └─README.md
```

## 使用方式

1. chrome浏览器打开  `chrome://extensions/`
2. 把 `layui-chrome.crx` 直接拖入
